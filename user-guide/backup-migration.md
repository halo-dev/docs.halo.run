# 备份

## 数据备份

目前 Halo 在后台的小工具中提供了数据导出的功能，此功能的作用为导出数据库的所有数据，格式为 `JSON`。通常可以作为切换数据库类型的时候使用。需要注意的是，此备份仅仅为备份数据，不包含其他诸如主题、附件等资料。如下图：

![halo-data-export.png](/assets/halo-data-export.png)

点击右下角的备份按钮即可导出所有数据，之后点击备份文件的标题即可下载。
